# Palamuti Luxe Craft

Palamuti Luxe Craft is a modern jewelry showcase website designed to present handcrafted jewelry collections with an elegant and responsive user interface.

## Project Overview

This website allows users to explore premium jewelry collections through a clean and visually appealing layout. The design focuses on simplicity, responsiveness, and a smooth browsing experience across devices.

## Features

- Responsive and modern UI design
- Product showcase for jewelry collections
- Image gallery for product display
- Smooth navigation and clean layout
- Optimized for desktop and mobile devices

## Technologies Used

The project is built using modern frontend technologies:

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Installation

Follow these steps to run the project locally.

### 1. Clone the repository

```bash
git clone <YOUR_GIT_URL>
```

### 2. Navigate to the project folder

```bash
cd <YOUR_PROJECT_NAME>
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

After running the server, open your browser and visit:

```
http://localhost:5173
```
